# Lenovo DDR5 Order Dispute (Canada)

This repository documents a consumer purchase dispute involving a delayed/non-delivered Lenovo Canada order for DDR5 memory and related communications. It is intended to help others understand the timeline, documents involved, and general legal concepts that commonly arise in non-delivery / cancellation-pressure scenarios.

> Personal information has been redacted. Nothing here is legal advice.

## What’s included

- `orders/`  
  Redacted order confirmations.

- `communications/`  
  Redacted communications with Lenovo (updates, arbitration opt-out correspondence, dispute notice).

- `contracts/`  
  Vendor terms referenced in the dispute (sales agreement, arbitration agreement).

- `legal/analysis.md`  
  High-level summary of Canadian contract-law concepts relevant to non-delivery, repudiation, remedies, and damages (informational only).

## Timeline (high level)

- Order placed → delivery window provided
- Delivery missed / delay communicated
- Vendor requested cancellation / offered refund
- Dispute notice issued and record compiled

(See the PDFs in `orders/` and `communications/` for dated artifacts.)
